# CONCISCON

CONCISCON is a web application designed to describe controls, safeguards, functions, and implementation groups related to an identified risk entered by the user. The system dynamically filters results based on letter-matching input, allowing efficient and intuitive risk analysis.

## Features

- User input for identified risks
- Description of:
  - Controls
  - Safeguards
  - Security functions
  - Implementation groups
- Dynamic filtering based on matching letters
- Responsive and user-friendly interface

## Technologies

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js
- **Architecture:** Client-server structure

## Installation
1. Clone the repository:
```
git clone https://github.com/DorisC-debug/CONCISCON.git
```
2. Install dependencies:
```
npm install
```

3. Run the application:
```
npm start
```
